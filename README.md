## Customer Certificates

### Description
A CRUD app build using Node, Express and PostGreSQL for the backend, and React and Redux for the front end. 

### To Install and Run

In the root folder,
1. run 'npm run initialize'
2. run 'npm start'
3. Go to localhost:3001
